---------------------------------------------------------------------------------
1. ВЛОЖЕННОСТЬ АМПЕРСАНДЫ
---------------------------------------------------------------------------------

.container{
    какойнибудь css;
    какойнибудь css;
    какойнибудь css;
    
    &__title{
     какойнибудь css;
     какойнибудь css;

     &_active{
        какойнибудь css;
     }
    }
}

---------------------------------------------------------------------------------
преобразовывается в 
---------------------------------------------------------------------------------

.container{
    какойнибудь css;
    какойнибудь css;
    какойнибудь css;
}

container__title{
    какойнибудь css;
    какойнибудь css;
}

container__title_active{
    какойнибудь css;
}

---------------------------------------------------------------------------------

& - преобразовывает имя родителя в этот символ(амперсанда)

---------------------------------------------------------------------------------




---------------------------------------------------------------------------------
2. ВЫДАЧА ПЕРЕМЕННОЙ $
---------------------------------------------------------------------------------

$название-переменной: что-то;
$main-color: blue;

color: $main-color;

---------------------------------------------------------------------------------
в CSS это (--var) 
---------------------------------------------------------------------------------




---------------------------------------------------------------------------------
3. ВЫДАЧА БОЛЬШОЙ ПЕРЕМЕННОЙ @mixin
---------------------------------------------------------------------------------

@mixin main-text{
    color:black;
    font-size: 20px;
}

---------------------------------------------------------------------------------
использование выглядит примерно так (сразу поставит все свойства)
---------------------------------------------------------------------------------

container{
    какойнибудь css;
    @include main-text;
}
